# Introduction

The Optimus API is a single REST API for **customers, wallets, KYC,
transactions, and investments**. You use it to launch branded wallet and
savings products for your customers — Optimus handles the ledger, compliance
checks, and money movement underneath.

All requests are made over HTTPS, authenticated with a secret key, and return
JSON. Request and response bodies use `snake_case` field names.

## Base URLs

Every endpoint in the [API reference](/docs/api-reference) is relative to one
of two base URLs:

## Make your first request

Grab a test secret key from [API keys](/api-keys) in the dashboard, then list
your customers:

<CodeBlock
  lang="bash"
  label="cURL"
  code={`curl ${API_SANDBOX_BASE_URL}/customers \\
  -H "Authorization: Bearer sk_test_your_secret_key"`}
/>

A successful response returns `200 OK` with a JSON body:

```json
{
  "status": true,
  "message": "Customers retrieved",
  "data": [],
  "meta": { "page": 1, "per_page": 25, "total": 0 }
}
```
